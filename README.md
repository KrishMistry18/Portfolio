## 👨‍💻 About

This is the source code for my personal developer portfolio. It serves as a living showcase of my engineering journey — from building on-device ML models in Flutter to creating 3D geopolitical dashboards in Next.js. The portfolio is designed to be fully responsive, accessible, and fast, highlighting my latest projects, technical skills, and professional experience.

## 🚀 Getting Started

To get a local copy up and running, follow these simple steps.

### Prerequisites

You only need a modern web browser. No complex build tools or package managers are required.

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/KrishMistry18/Portfolio.git
   ```
2. Navigate into the project directory:
   ```bash
   cd Portfolio
   ```
3. Open `index.html` in your favorite browser.
