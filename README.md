# Portfolio 🚀

**🔗 Live:** https://portfolio-eta-pied-17.vercel.app/

Personal portfolio website showcasing projects, skills, and experience — built as a fast, responsive static site.

---

## About

This is the source code for my personal developer portfolio. It highlights my work across AI systems, full-stack web development, and mobile applications, and serves as a living record of my growth as an engineer.

---

## Tech Stack

| Layer | Technology |
|---|---|
| Frontend | HTML5, CSS3, JavaScript |
| Deployment | Vercel |

---

## Getting Started

```bash
git clone https://github.com/KrishMistry18/Portfolio.git
cd Portfolio
```

Open `index.html` in your browser, or deploy to any static host.

---

## Deployment

The site is deployed on Vercel. Any static host works (Netlify, GitHub Pages, Cloudflare Pages).
