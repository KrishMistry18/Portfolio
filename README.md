## About

This is the source code for my personal developer portfolio. It serves as a living showcase of my engineering journey — from building on-device ML models in Flutter to 3D geopolitical dashboards in Next.js.

## Getting Started

```bash
git clone https://github.com/KrishMistry18/Portfolio.git
cd Portfolio
```

Open `index.html` in your browser — no build step required.

## Deployment

Deployed on Vercel. Any static host works — Netlify, GitHub Pages, Cloudflare Pages.
