ImpactGlobe

Live Web ↗
For traders & analysts who need geopolitical context without digging through 20 news tabs — a live intelligence layer that maps world events directly to Forex impact.
  • 3D globe (Three.js + Leaflet) mapping live geopolitical events with severity tiers (Critical, High, Medium) and historical 24-hour replay.
  • Gemini 2.0 Flash auto-generates structured news intelligence with direct Forex impact analysis (e.g., EUR/USD, XAU/USD).
  • Real-time event tracking and live market panel powered by Next.js and Firebase — zero manual refresh needed.

⚙️ Hardest Technical Decision

The trickiest challenge was syncing live geopolitical event data with Three.js globe rendering without causing frame drops. Solved it by debouncing Firebase listeners and batching DOM updates outside the animation loop.

HemiSphere

Live Web ↗
Urban communities lack fast, low-cost tools to report and act on safety hazards. HemiSphere puts AI-powered incident classification and CO₂ tracking directly in residents' hands — no cloud dependency needed.
  • MobileNetV3 CNN (Transfer Learning) classifies accidents, construction hazards, and garbage from incident photos with on-device inference.
  • Heuristic regression for real-time CO₂ emissions tracking — gives communities actionable environmental data locally.
  • Flutter + Firebase Firestore backend with real-time sync across all community members.

⚙️ Hardest Technical Decision

The key decision was choosing on-device inference (MobileNetV3) over a cloud API — this eliminated latency, worked offline, and kept sensitive incident photos off external servers. The tradeoff was a larger app bundle, offset with lazy loading.

ChessAnalyzer

Live Web ↗
Chess players improving their game need engine analysis, but Lichess and Chess.com require accounts and uploads. ChessAnalyzer runs full Stockfish analysis entirely in the browser — no backend, no sign-up, instant feedback.
  • Play-vs-Stockfish with adjustable strength, multi-engine support with WASM threads gracefully degrading based on CPU.
  • Move classification, multi-PV engine lines, and game summaries highlighting inaccuracies, mistakes, and blunders.
  • 40+ piece themes, SAN move list, zero backend dependency for analysis.
