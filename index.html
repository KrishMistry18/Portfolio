ImpactGlobe — Geopolitical Intelligence Dashboard

A real-time intelligence platform on an interactive 3D globe. Track geopolitical events, live forex markets, environmental overlays, and AI-generated impact analysis — all in one dashboard.

  • Live forex panel powered by Twelve Data API and Supabase Realtime push — zero manual refresh needed
