Outside code, I'm a national-level chess competitor — which has given me an obsessive appreciation for long-horizon thinking and knowing when to sacrifice short-term for strategic gain. That mindset shows up in how I architect systems.

ImpactGlobe

Live Web ↗
For traders & analysts who need geopolitical context without digging through 20 news tabs — a live intelligence layer that maps world events directly to Forex impact.

  • Gemini 2.0 Flash auto-generates structured news intelligence with direct Forex impact analysis (e.g., EUR/USD, XAU/USD).
  • Real-time event tracking and live market panel powered by Next.js and Firebase — zero manual refresh needed.

HemiSphere

Live Web ↗
Urban communities lack fast, low-cost tools to report and act on safety hazards. HemiSphere puts AI-powered incident classification and CO₂ tracking directly in residents' hands — no cloud dependency needed.
  • MobileNetV3 CNN (Transfer Learning) classifies accidents, construction hazards, and garbage from incident photos with on-device inference.
  • Heuristic regression for real-time CO₂ emissions tracking — gives communities actionable environmental data locally.
  • Flutter + Firebase Firestore backend with real-time sync across all community members.
Flutter TensorFlow MobileNetV3 Firebase Python

ChessAnalyzer

Live Web ↗
Chess players improving their game need engine analysis, but Lichess and Chess.com require accounts and uploads. ChessAnalyzer runs full Stockfish analysis entirely in the browser — no backend, no sign-up, instant feedback.
  • Play-vs-Stockfish with adjustable strength, multi-engine support with WASM threads gracefully degrading based on CPU
  • Move classification, multi-PV engine lines, and game summaries highlighting inaccuracies, mistakes, and blunders
  • 40+ piece themes, SAN move list, zero backend dependency for analysis
Next.js 15 TypeScript Stockfish WASM MUI Jotai
GitHub Repo ↗

EduCycle

Live Web ↗
Students spend thousands on textbooks that sit idle after a semester. EduCycle is a verified peer-to-peer marketplace where students buy, sell, and recycle academic materials — with real payments and secure auth built in.
  • JWT + OAuth2 auth, 2FA, real-time chat, and Stripe payment integration with an analytics dashboard
  • Django REST + React.js backend; Redis caching, advanced search and filtering
  • PWA-enabled app with offline support, and a fully responsive UI
Django REST React.js PostgreSQL Redis Stripe

HeritageGuide

Live Web ↗
India's heritage sites are underdocumented and hard to plan around. HeritageGuide gives travellers an interactive map, smart multi-modal routing, and a slot-based booking system — all in one open-source tool.
  • Interactive Leaflet + OSM map with smart multi-modal routing (driving/walking/cycling/transit) via Leaflet Routing Machine
  • Multi-day itinerary planner, slot-based attraction booking with availability enforcement
  • Optional Google Maps upgrade for Place Search and Drawing tools
Django 5 Leaflet OpenStreetMap Google Maps API Firebase

AyurWell

Live Web ↗
Ayurvedic health advice is buried in books and expensive consultations. AyurWell makes it accessible — a dosha assessment engine and AI-generated diet plans that adapt to your body type and season.
  • Dosha assessment engine with Vata / Pitta / Kapha scoring and AI-generated seasonal diet plans
  • Django REST backend with token-based auth, CORS, and audited login/signup event logging
  • Progress dashboard tracking energy, sleep, hydration trends via Recharts visualisations
Vite + React TypeScript Django REST shadcn-ui Tailwind

WatchParty

Watch-together services like Teleparty require browser extensions and Netflix subscriptions. WatchParty works with YouTube, needs zero installs, and syncs video state across all participants in under a second using pure WebRTC.
  • Sub-second video sync via Socket.IO signalling + YouTube IFrame API state reconciliation
  • WebRTC data channels for P2P chat, voice, and screen sharing — no media server needed
  • Room system with configurable capacity (2–10), video queue, emoji reactions, and chat history
Node.js Express.js WebRTC Socket.IO YouTube API

Nexus Financial

Built as a Java DSA project that became a real banking simulation — Nexus Financial demonstrates ledger-accurate ATM operations, SQL injection prevention, and a full audit trail without any third-party dependencies.
  • Ledger-derived balance (sum of deposits − withdrawals) eliminates duplication and inconsistency
  • Full ATM ops: Deposit, Withdrawal, Fast Cash, Balance Enquiry, Mini Statement, PIN Change
  • SQL injection prevention via JDBC prepared statements; dual-table transaction audit trail
